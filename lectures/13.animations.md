--- VERTICAL SLIDE ---

Blending poses requires matrix interpolation.

```
Matrix4x4 InterpolateMatrices(float alpha,
    const Matrix4x4& first, const Matrix4x4& second)
{
    return alpha * first + (1 - alpha) * second;
}
```
